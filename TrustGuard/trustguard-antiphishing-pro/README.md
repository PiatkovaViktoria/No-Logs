# TrustGuard - Антифишинг PRO

Расширение для Google Chrome: проверка ссылок по локальной базе, Google Safe Browsing и VirusTotal. Защита в Gmail и Mail.ru, счётчик блокировок, страница-заглушка с кнопками «Назад» и «Всё равно перейти».

## Установка

1. Откройте `chrome://extensions/`.
2. Включите **Режим разработчика**.
3. Нажмите **Загрузить распакованное расширение** и выберите папку с расширением.
4. Готово — расширение активно.

## Что делает

- **Перехват переходов** — перед загрузкой страницы проверяет URL по трём источникам:
  - локальная база (20+ примеров в `phishing-list.json`);
  - Google Safe Browsing API (нужен API-ключ в настройках);
  - VirusTotal API (нужен API-ключ; блокировка при риске > 70%).
- **Блокировка** — при опасном сайте открывается страница-заглушка с причиной блокировки и кнопками «Назад» и «Всё равно перейти».
- **Счётчик** — число заблокированных переходов хранится в `storage` и отображается в попапе и на странице блокировки.
- **Gmail и Mail.ru** — на этих сайтах скрипт забирает отправителя, тему, текст и ссылки из открытого письма, отправляет на проверку в background и показывает метку (зелёная / жёлтая / красная) с результатом.
- **Попап** — счётчик «Заблокировано: X», статус защиты, последние заблокированные сайты, последние проверенные письма с оценкой риска, вкладка «Настройки» для ввода API-ключей.

## API-ключи (необязательно)

- **Google Safe Browsing**: [Google Cloud Console](https://console.cloud.google.com/) → APIs & Services → включите Safe Browsing API → создайте ключ.
- **VirusTotal**: [VirusTotal](https://www.virustotal.com/gui/my-apikey) → профиль → API key.

Без ключей работает только проверка по локальной базе из `phishing-list.json`.

## Файлы

| Файл | Назначение |
|------|------------|
| `manifest.json` | Манифест (Manifest V3), разрешения, content_scripts для Gmail и Mail.ru. |
| `background.js` | Проверка по трём источникам, редирект на block.html, счётчик, последние блокировки/письма, обработка «всё равно перейти». |
| `block.html`, `block.js` | Страница блокировки: причина, URL, кнопки «Назад» и «Всё равно перейти». |
| `content-mail.js` | Внедрение в Gmail и Mail.ru: извлечение данных письма и ссылок, запрос к background, метка по результату. |
| `popup.html`, `popup.js` | Попап: счётчик, статус, последние сайты и письма, настройки API. |
| `phishing-list.json` | Локальный список фишинговых доменов (можно редактировать). |

Код с комментариями на русском в ключевых местах.
